# NextJs Podcast
## 🚀 Live link - [NextPodcast](https://prodcust-app.vercel.app/)

# 🪶 Features
- Used NextAuth for authentication
- Auto audio play in Podcast details page
- Users can adjust the current playback time of the audio using a range slider

 - Next js
 - Tailwind
 - CSS
 - Redux Tool-kit
